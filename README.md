# 代码世界

此仓库是一个用于 HTML、CSS、JavaScript 和 Python 项目的学习练习场。

## 目录结构

- `HTML/` — 52 个独立前端练习：表单、布局、动画、响应式页面
- `JavaScript/` — 46 个练习和小型应用：算法、交互式 UI、实用工具
- `Python/` — 13 个算法与数据结构练习
- `SQL/` — 数据库练习（students.sql, universe.sql）

## 使用说明

### HTML
在浏览器中打开任意 `.html` 文件，或使用 Live Server 扩展获得更好的开发体验。

### JavaScript
```powershell
cd "d:\代码世界\Learning-World\JavaScript"
node <filename>.js        # 脚本
node app.js              # 服务端应用（需要 Node.js 服务器）
```

### Python
```powershell
cd "d:\代码世界\Learning-World\Python"
python <filename>.py
```

### SQL
使用数据库客户端执行 `students.sql` 或 `universe.sql`。

## 文件命名

使用描述性名称，以连字符分隔（如 `cafe-menu.html`）。
